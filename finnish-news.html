Uutisviikko selkosuomeksi - Iranin sota, ydinaselaki, puoluekannatus, naistenpäivä

本周主题：伊朗战争、芬兰核武法律、政党支持率、国际妇女节

📝 新闻摘要

本周新闻主要有4个主题：1）伊朗战争持续超过一周，美国袭击伊朗境内数千个目标；2）芬兰政府宣布修改核武法律，允许在国防需要时运输核武器到芬兰；3）芬兰政党支持率调查：SDP 25%领先，Kokoomus 18%；4）国际妇女节当天赫尔辛基有2万人参加游行。

💬 主题一：伊朗战争 (Iranin sota)

Tällä viikolla uutisissa on seurattu joka päivä Iranin sotaa.

本周新闻每天都在报道伊朗战争。

  • Tällä viikolla = 本周
  • uutisissa = 在新闻中 (uutiset 的inessive复数)
  • on seurattu = 已经被追踪/报道 (被动语态)
  • joka päivä = 每天
  • Iranin sota = 伊朗战争

Iranin sota on jatkunut nyt yli viikon.

伊朗战争现在已经持续超过一周。

  • on jatkunut = 已经持续 (jatkua 的完成时)
  • nyt = 现在
  • yli = 超过
  • viikon = 一周 (viikko 的宾格)

USA kertoo, että se on iskenyt tuhansiin kohteisiin Iranissa.

美国表示已经袭击了伊朗境内的数千个目标。

  • kertoo = 告诉/说 (第三人称单数)
  • ettei = (后面的从句) 因为用 että
  • se = 它 (指美国)
  • on iskenyt = 已经袭击 (iskeä 的完成时)
  • tuhansiin = 数以千计的 (tuhat 的复数部分格)
  • kohteisiin = 目标 (kohde 的复数部分格)
  • Iranissa = 在伊朗 (伊朗的属格 +ssa/:ssa)

USA on iskenyt esimerkiksi sotilaskohteisiin ja öljylaitoksiin.

例如，美国袭击了军事目标和炼油厂。

  • esimerkiksi = 例如
  • sotilaskohteisiin = 军事目标 (sotilas + kohde 复合词)
  • öljylaitoksiin = 炼油厂 (öljy + laitos 复合词)

Iran on tehnyt vastaiskuja naapurimaihin, joissa on USA:n tukikohtia.

伊朗对邻国发动了反击，这些国家有美国的军事基地。

  • on tehnyt = 已经做/发动 (tehdä 的完成时)
  • vastaiskuja = 反击 (vastaisku 的复数部分格)
  • naapurimaihin = 邻国 (naapurimaa 的复数部分格)
  • joissa = 在其中 (joka 的复数inessive)
  • tukikohtia = 基地 (tukikohta 的复数部分格)
  • USA:n = 美国的 (USA 的属格)

Iskuja on ollut jo 12 maassa Lähi-idässä.

在中东的12个国家已经发生了袭击事件。

  • Iskuja = 袭击 (isku 的复数主格)
  • on ollut = 已经有 (olla 的完成时)
  • jo = 已经
  • 12 maassa = 在12个国家 (maa 的复数inessive)
  • Lähi-idässä = 在中东 (Lähi-itä 的inessive)

Lähi-idän maissa on noin 3000 suomalaista.

中东地区大约有3000名芬兰人。

  • Lähi-idän maissa = 在中东国家 (Lähi-idän = 中东的，maissa = 在国家里)
  • noin = 大约
  • suomalaista = 芬兰人 (suomalainen 的部分格)

Monet heistä haluavat tulla sodan takia Suomeen.

他们中的许多人因为战争想要来到芬兰。

  • Monet = 许多 (moni 的复数主格)
  • heistä = 他们其中 (he 的复数部分格)
  • haluavat = 想要 (haluta 的第三人称复数)
  • tulla = 来 (tulla 的不定式)
  • sodan takia = 因为战争 (sota 的属格 + takia)
  • Suomeen = 到芬兰 (Suomi 的进入格)

Lähi-idästä on kuitenkin vaikea päästä matkustamaan pois, koska lentoliikenne ei toimi.

然而，由于航班停飞，很难从中东离开。

  • Lähi-idästä = 从中东 (Lähi-idä 的离开格)
  • kuitenkin = 然而
  • on vaikea = 很难 (形容词 + 位)
  • päästä = 能够/允许 (päästä 的不定式)
  • matkustamaan = 去旅行 (matkustaa 的动名词)
  • pois = 离开
  • koska = 因为
  • lentoliikenne = 航班/航空交通
  • ei toimi = 不运行/停飞

Ulkoministeriö järjesti lentokoneen hakemaan suomalaisia, mutta lennolle mahtui vain noin 160 ihmistä.

外交部安排了一架飞机接回芬兰人，但航班只能容纳约160人。

  • Ulkoministeriö = 外交部
  • järjesti = 安排 (järjestää 的过去时)
  • lentokoneen = 飞机 (lentokone 的宾格)
  • hakemaan = 接回/寻找 (hakea 的动名词)
  • suomalaisia = 芬兰人 (suomalainen 的复数部分格)
  • mutta = 但是
  • lennolle = 航班 (lento 的方向格)
  • mahtui = 容纳 (mahtua 的过去时)
  • vain = 只
  • ihmistä = 人 (ihminen 的部分格)

⚛️ 主题二：芬兰核武法律 (Suomen ydinaselaki)

Hallitus kertoi tällä viikolla, että Suomi muuttaa lakia ydinaseista.

政府本周宣布芬兰将修改核武器法律。

  • Hallitus = 政府
  • kertoi = 宣布/说 (kertoa 的过去时)
  • Suomi = 芬兰
  • muuttaa = 改变
  • lakia = 法律 (laki 的部分格)
  • ydinaseista = 核武器 (ydinase 的复数部分格)

Puolustusministeri Antti Häkkänen kertoi, että jatkossa Suomen alueelle voidaan kuljettaa ydinase, jos sitä tarvitaan Suomen puolustamiseen.

国防部长Antti Häkkänen表示，今后如果需要用于芬兰的防御，可以将核武器运送到芬兰领土。

  • Puolustusministeri = 国防部长
  • jatkossa = 今后/从现在起
  • alueelle = 到地区/领土 (alue 的方向格)
  • voidaan = 可以 (voida 的被动语态)
  • kuljettaa = 运输
  • jos = 如果
  • sitä = 它 (se 的部分格)
  • tarvitaan = 需要 (tarvita 的被动语态)
  • puolustamiseen = 为了防御 (puolustaminen 的方向格)

Häkkänen sanoo, että lakimuutos tarvitaan, koska Suomi on nyt Naton jäsen.

Häkkänen说，法律修改是必要的，因为芬兰现在是北约成员国。

  • sanoo = 说 (sanoa 的第三人称单数)
  • lakimuutos = 法律修改
  • tarvitaan = 需要
  • koska = 因为
  • Naton jäsen = 北约成员

Natomaissa ei yleensä ole täyttä kieltoa ydinaseille.

北约国家通常没有完全禁止核武器。

  • Natomaissa = 在北约国家 (Nato-maa 的复数inessive)
  • ei yleensä = 通常不
  • ole = 是 (olla 的否定形式)
  • täyttä = 完全的 (täysi 的部分格)
  • kieltoa = 禁止 (kielto 的部分格)
  • ydinaseille = 对核武器 (ydinase 的与格复数)

Oppositiopuolueet ovat yllättyneet hallituksen ilmoituksesta, että ydinaseen tuonti sallitaan Suomeen.

反对党对政府宣布允许核武器进口到芬兰感到惊讶。

  • Oppositiopuolueet = 反对党 (oppositiopuolue 的复数主格)
  • ovat yllättyneet = 感到惊讶 (yllättyä 的完成时)
  • ilmoituksesta = 宣布/通知 (ilmoitus 的离开格)
  • tuonti = 进口/进入
  • sallitaan = 被允许 (sallia 的被动语态)

Asiaa on valmisteltu hallituksessa hiljaa.

此事在政府内部一直是秘密准备的。

  • Asiaa = 事情/议题 (asia 的部分格)
  • on valmisteltu = 已经被准备 (valmistella 的被动完成时)
  • hiljaa = 安静地/秘密地

🗳️ 主题三：政党支持率 (Puoluekannatus)

Yle julkaisi tällä viikolla taas puolueiden kannatuskyselyn.

Yle本周再次发布了政党支持率调查。

  • julkaisi = 发布 (julkaista 的过去时)
  • taas = 再次
  • puolueiden = 政党 (puolue 的复数属格)
  • kannatuskyselyn = 支持率调查

Suosituin puolue on edelleen SDP.

最受欢迎的仍然是社会民主党（SDP）。

  • Suosituin = 最受欢迎的 (suosittu 的最高级)
  • edelleen = 仍然

SDP:tä kannattaa noin 25 prosenttia suomalaisista.

大约25%的芬兰人支持SDP。

  • SDP:tä = SDP (SDP 的部分格)
  • kannattaa = 支持 (政党)
  • noin = 大约
  • prosenttia = 百分之...
  • suomalaisista = 芬兰人 (suomalainen 的复数部分格)

Toiseksi suosituin puolue on kokoomus.

第二受欢迎的政党是民族联合党（Kokoomus）。

  • Toiseksi = 第二 (序数词)
  • kokoomus = 民族联合党

Kolmanneksi suosituin puolue on hallituspuolue perussuomalaiset.

第三受欢迎的政党是执政党芬兰人党。

  • Kolmanneksi = 第三
  • hallituspuolue = 执政党/政府党
  • perussuomalaiset = 芬兰人党

👩 主题四：国际妇女节 (Kansainvälinen naistenpäivä)

Viikon päätteeksi on vietetty kansainvälistä naistenpäivää tänään sunnuntaina.

本周结束时，今天周日庆祝了国际妇女节。

  • Viikon päätteeksi = 本周结束时
  • on vietetty = 已经被庆祝 (viettää 的被动完成时)
  • kansainvälistä = 国际的 (kansainvälinen 的部分格)
  • naistenpäivää = 妇女节 (naistenpäivä 的部分格)

Helsingissä järjestettiin naistenpäivän marssi.

赫尔辛基举办了妇女节游行。

  • Helsingissä = 在赫尔辛基
  • järjestettiin = 被举办 (järjestää 的被动过去时)
  • naistenpäivän = 妇女节的 (naistenpäivä 的属格)
  • marssi = 游行

Marssille osallistui noin 20 000 ihmistä.

大约2万人参加了游行。

  • Marssille = 对游行 (marssi 的方向格)
  • osallistui = 参加 (osallistua 的过去时)
  • ihmistä = 人 (ihminen 的部分格)

Marssilla vaadittiin, että tasa-arvoa pitää lisätä.

游行要求增加性别平等。

  • Marssilla = 在游行上
  • vaadittiin = 被要求 (vaatia 的被动过去时)
  • tasa-arvoa = 性别平等
  • pitää = 必须
  • lisätä = 增加
